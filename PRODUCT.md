# Product

## Register

product

## Users

Hosts (educators, enterprise trainers, event organizers) who run live webinars and workshops. They need a clear at-a-glance view of session health, engagement, and attendee response data — typically viewed on desktop in a focused work context.

## Product Purpose

IntelliMeet is a live session management platform for hosts to create, launch, and analyze interactive webinars and workshops. The host dashboard is the command center: monitor sessions, track attendee engagement, and manage session libraries.

## Brand Personality

Sharp, capable, trustworthy. Premium but approachable — the tool that disappears into the task.

## Anti-references

- Flat SaaS dashboards that feel like spreadsheets
- Over-animated, distracting enterprise dashboards
- Cream/warm-tinted backgrounds (the AI default)

## Design Principles

1. The interface serves the session — clarity over decoration
2. State is always legible — live, ended, empty all look intentionally different
3. Hierarchy earns its place — important numbers are big, support text is small
4. Actions are obvious — primary CTAs are always visible and reachable

## Accessibility & Inclusion

WCAG AA minimum. Reduced-motion respected. Body text must hit ≥4.5:1 contrast.
